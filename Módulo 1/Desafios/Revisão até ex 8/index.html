Tag

Podemos abrir uma caixa de texto em um parágrafo utilizando a tag <p>

Podemos determinar os niveis de tamanho dos titulos do seu site, utilizando a tag <h1>,<h2>,<h3>,<h4>, e assim por diante até o nivel 6

Podemos colocar uma linha de divisão em nosso texto utilizando a tag <hr>

Podemos quebrar uma linha ou mais utilizando a tag <br>, como mostrado abaixo



Podemos também adicionar comentários que não vão aparecer no site apenas no código, como mostrado a seguir

Podemos adicionar simbolos de verificação, utilizando diversas tags como mostrado a seguir

® © ™ e diversas outras da lista em PDF

Podemos também adicionar imagens aos nossos sites utilizando a tag <img>, como demostrado abaixo

Podendo ou não ser baixado no PC e colocada na pasta do arquivo junto ao código, ou como demonstrado, utilizando a própria URL da imagem



Podemos também add Faviocons utilizando a tag <link>, acima de title, no inicio do código

Podemos também formatar nosso site, com cor da página, tamanhos de fontes, cor das palavras, e etc, utilizando a tag <style>, podendo ser colocada acima de head para englobar o site em geral, ou utilizada como mostrado a seguir, digitando em um local expecifico dentro de alguma tag e ultilizando um parâmetro como "background-color"

Podemos colocar um texto em negrito, utilizando a tag <strong>(forte)

Podemos colocar um texto em itálico utilizando a tag <em>

Podemos indicar que um texto é um endereço utilizando a tag <address>,

Rua Faveiro 1021 - FRG - PR

Podemos colocar um texto em destaque como um marca texto utilizando a tag <mark>, outro exemplo sem utilizar o parametro style na própria tag

Podemos colocar um texto com letras maiores para destaque utilizando a tag <big>

Podemos colocar um texto com letras miudas utilizando a tag <small>

Podemos colocar um texto com a letra riscada signifando exclusão utilizando a tag <del>

Podemos colocar um texto como sublinhado significando destaque, utilizando a tag <ins>

Podemos colocar um texto sobrescrito como no ex: x20+3, utilizando a tag <sup>

Podemos colocar um texto subscrito como no ex: H2O, utilizando a tag <sub>

OBS: Foi ensinado também a colocar emoji no site, da para copiar o código do emoji ou copiar e colar o próprio emoji, mas em alguns navegadores isso não é possivel, mas lembrando que dá pra fazer.
